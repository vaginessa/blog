Argumentos da linha de comandos

Argumentos da linha de comandos:
-page $pageno  abrir documento na página especificada
-named-dest $dest  abrir documento existente no diretório especificado
-lang $language  definir o idioma da interface gráfica. $language é o código com duas letras (ex:"en", "pt")
-restrict  executar aplicação no modo restrito.
-reuse-instance  utilizar a instância aberta ao invés de iniciar um novo processo
-presentation $file.pdf  abrir um documento no modo de apresentação (ecrã completo)
-register-for-pdf  forçar o registo do Sumatra PDF como leitor de PDFs do sistema
-esc-to-exit  se indicada, a tecla ESC termina o programa. A opção é memorizada no ficheiro de definições. Assim só necessita de ser indicada uma vez
-bg-color $color  altera a cor de fundo da aplicação para a cor especificada, desde que o seja no formato hex (ez: 0xffff00)
-invert-colors  inverte as cores do documento (ex: a cor preta fica branca e a cor branca fica preta).

Nota: as opções -esc-to-exit e -bg-color alteram as omissões e ficam gravadas no ficheiro sumatrapdfprefs.dat (em %appdata%\SumatraPDF se a aplicação for instalada e caso utilize o modo portátil, no mesmo diretório do executável). Para recuperar as opções omissas, pode apagar o ficheiro sumatrapdfprefs.dat.

Relação entre a impressão e a linha de comandos:

-print-to-default $file.pdf  imprime o documento para a impressora padrão
-print-to $printer_name $file.pdf  imprime o documento para a impressora indicada
-print-dialog $file.pdf  abre o documento e exibe a janela de impressão para que possa escolher a impressora e as opções
-exit-on-print  se utilizar -print-to-default ou -print-to, após a impressão a aplicação será fechada

Suporte a LaTeX

O Sumatra possibilita a visualização dos documentos LaTeX. Visite esta página para mais informações e instruções para configurar os diversos editores de texto de modo a que possa utilizar esta funcionalidade.

Execução no modo restrito

Este modo é particularmente útil caso pretenda utilizar SumatraPDF como o visualizador incorporado para a documentação do seu programa ou no modo "kiosk". Neste modo existem diversas operações que não são permitidas:

  • abrir novos ficheiros
  • abrir as URLs existentes nos documento PDF
  • selecionar texto e imagens
  • impressão
  • alteração das definições
  • gravar documentos no disco
  • verificação automática e manual de atualizações
  • ver histórico dos ficheiros visualizados
  • suporte a documentos TeX
  • registar aplicação como o leitor de PDF
  • abrir ficheiros com o Adobe Acrobat
  • enviar documentos por mensagem eletrónica

Notas

Se "Lembrar definições para cada documento" estiver ativo e o documento for visualizado noutra ocasião, o Sumatra PDF irá abrir o documento na página em que estava quando o fechou. As definições não são exclusivas dos documentos exibidos no menu "Ficheiro" - no entanto, se mover o documento para outra pasta, o SumatraPDF não vai conseguir recordar as definições e estas não serão utilizadas.

Algumas definições (ex: ecrã completo) são gravadas como omissão se não utilizar uma opção na interface gráfica e caso o SumatraPDF seja fechado sem qualquer documento aberto.

Limitações identificadas

A edição de formulários e adição de comentários não está implementada.

Desativar a visualização de documentos PDF no seu navegador web (IE e Firefox)

O Adobe Acrobat possui um "plug-in" para o Internet Explorer e para o Firefox que possibilita a visualização de documentos PDF diretamente no seu navegador web.

Nem todos os utilizadores gostam desta opção e você tem a possibilidade de utilizar o SumatraPDF como o visualizador de documentos PDF, tanto no IE como no Firefox, sem ter que desinstalar o Adobe Acrobat.

Aqui estão os procedimentos necessários (baseado nesta publicação):

  • Iniciar Adobe Acrobat
  • Selecionar Editar/Preferências
  • Selecionar Internet
  • Desmarcar a opção "Exibir PDF no navegador"
  • Selecionar OK
  • Iniciar o Sumatra PDF
  • Selecionar Definições/Opções... e escolha "Tornar SumatraPDF o leitor de PDFs omisso"

Agora, ao clicar nas ligações que contenham documentos PDF, estes já não devem ser abertos no navegador e por sua vez deve aparecer uma janela de diálogo. Selecione "Utilizar sempre esta opção..." caso pretenda que os documentos sejam automaticamente abertos no SumatraPDF.
