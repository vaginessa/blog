Liste de lecteurs et visionneuses ePub pour Windows

lecteurs et visionneuses ePub

Sumatra PDF

Sumatra PDF supporte l'affichage des e-books ePub depuis sa version 2.1. Sumatra is gratuit, open-source et supporte plusieurs autres formats, tels que PDF, CHM, MOBI, XPS, DjVu, CBR, CBZ.

FBReader

FBReader est un lecteur de e-Books multi-plateforme et open-source. Il peut aussi lire des fichiers ePub.

Mobipocket Reader

Mobipocket Reader supporte, parmi plusieurs autres, le format ePub.

Cool Reader

Cool Reader supporte le format ePub ainsi que quelques autres.

EPUBReader FireFox extension

EPUBReaderest une extension (un module) FireFox pour la lecture en ligne de fichier ePub.

Calibre

Calibre est un logiciel de gestion de bibliothèques avec une visionneuse intégrée permettant de lire / convertir de nombreux formats de fichiers, principalement ceux utilisés pour les e-Books.

Renseignements sur le format ePub

ePub est un format ouvert defini par le Open eBook Forum de l'International Digital Publishing Forum. Il est basé sur les formats XHTML, XML et CSS.

Les fichiers ePub ont habituellement l'extension .epub.
